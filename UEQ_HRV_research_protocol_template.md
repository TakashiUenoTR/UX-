# UEQ + HRV 研究用テンプレ（実験プロトコル付き）

------------------------------------------------------------------------

# 1. 研究概要

## 目的

UEQ（User Experience Questionnaire）とHRV（Heart Rate
Variability）を組み合わせ、
主観評価と生体反応を統合したUX評価を実施する。

### 主な研究質問（Research Questions）

-   UXの主観評価（UEQ）とHRV指標は相関するか？
-   特定のUI操作はストレス反応を引き起こすか？
-   Hedonic UX（刺激性・魅力）は生理指標に反映されるか？

------------------------------------------------------------------------

# 2. 実験デザイン

## 実験構成

    Baseline（安静）
       ↓
    Task A（UI操作）
       ↓
    Task B（比較条件）
       ↓
    UEQ回答

------------------------------------------------------------------------

## 被験者条件

-   年齢：\_\_\_\_\_\_
-   性別：\_\_\_\_\_\_
-   UX経験レベル：初心者 / 中級 / 上級
-   サンプル数目安：
    -   最低：15名
    -   推奨：30名以上

------------------------------------------------------------------------

# 3. 測定項目

## 主観評価

-   UEQ（standard または UEQ-S）

### UEQスケール

-   Attractiveness
-   Perspicuity
-   Efficiency
-   Dependability
-   Stimulation
-   Novelty

------------------------------------------------------------------------

## 生体計測

### 心拍数（HR）

用途：

-   興奮
-   緊張
-   集中

### HRV指標

  指標    意味
  ------- ----------------------------
  RMSSD   リラックス・副交感神経活動
  HF      回復・快適性
  LF/HF   ストレス・認知負荷

------------------------------------------------------------------------

# 4. 機材構成

## センサー

-   Polar H10（推奨）
-   HRV対応スマートウォッチ / スマートリング

## データ取得

-   BLE通信
-   Python（bleakなど）

## 可視化

-   pyqtgraph（リアルタイム表示）

------------------------------------------------------------------------

# 5. 実験手順

## STEP1：ベースライン測定

-   安静状態で3〜5分
-   呼吸を安定させる

取得：

-   HR
-   HRV

------------------------------------------------------------------------

## STEP2：UXタスク実施

例：

-   UI操作
-   新旧UI比較
-   ナビゲーションタスク

記録：

-   タスク開始・終了タイムスタンプ

------------------------------------------------------------------------

## STEP3：UEQ回答

重要：

-   タスク直後に実施
-   記憶バイアスを防ぐ

------------------------------------------------------------------------

# 6. データ同期

    12:01:30, 95, 12, error_event

# 8. 結果解釈ガイド

  状態                        解釈
  --------------------------- ------------
  HR↑ + Stimulation↑          良い興奮
  HR↑ + Efficiency↓           UXストレス

------------------------------------------------------------------------

# 9. UX統合評価（上級）

UX Quality Index（UXQI）例：

    UXQI =
      0.4 * UEQ_score +
      0.3 * Task_success +
      0.3 * HRV_relaxation_index

------------------------------------------------------------------------

# 10. 注意点（研究品質）

-   HRVは呼吸の影響を強く受ける
-   カフェイン・睡眠状態を記録
-   個人差が大きいため正規化推奨
